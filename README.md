# 🧑‍⚕️ AI Disease Prediction System

A machine learning-based web application that predicts diseases based on user-selected symptoms.

## 🚀 Features

* Predict diseases from symptoms
* Top-3 predictions with confidence scores
* Interactive and user-friendly UI
* Fast and lightweight model (Logistic Regression)
* Real-time predictions using Streamlit

## 🛠 Tech Stack

* Python
* Pandas
* Scikit-learn
* Streamlit

## 📊 How It Works

1. User selects symptoms
2. Model processes input using trained ML algorithm
3. System outputs most likely diseases with probabilities

## ▶️ Run Locally

```bash
pip install -r requirements.txt
python main.py
streamlit run app/app.py
```
